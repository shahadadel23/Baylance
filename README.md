# Baylance

AI-powered Personality Prediction and Intelligent Team Formation System.

## Overview
Baylance is an intelligent system that predicts MBTI personality types using Artificial Intelligence and Natural Language Processing (NLP), then automatically forms balanced teams.

## Features
- MBTI personality prediction
- Intelligent team formation
- Role assignment based on personality traits
- User, Supervisor, and Admin dashboards
- Task management system

## Technologies Used
- Flutter
- Firebase
- Python
- DistilBERT
- Hugging Face Transformers

## Results
- Accuracy: 96.8%
- F1-Score: 0.94 – 0.98
